# Developer Feed

A developer news reader that fetches articles from DEV.to and displays them as cards.

## Features
- Browse latest developer articles from DEV.to
- Click any article to read it fully within the app
- Bookmark articles and access them in your Reading List
- Sort by newest or oldest
- Filter by tag

## Requirements
- Node.js v18 or newer — https://nodejs.org
- pnpm — install with: `npm install -g pnpm`

## Getting Started

```bash
# 1. Install dependencies
pnpm install

# 2. Start the dev server
pnpm dev

# 3. Open in browser
# http://localhost:5173
```

## Build for Production

```bash
pnpm build
pnpm preview
```
